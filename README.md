## Getting Started

### Prerequisites

Before you begin, ensure you have the following installed:

-   Node.js
-   Expo CLI

### Installation

1. Clone the repository:

    ```bash
    git clone https://github.com/your-username/water-pal.git
    ```

2. Install dependencies:
    ```
    npm install
    ```
3. Run app:
    ```
    npx expo start
    ```
